## Descripción de proyecto

Sitio web para Peña Bros Upholstery, un taller de remodelación de autos con sede en San Antonio, Texas. El proceso fue una colaboración con [AllSource Marketing] (http://allsourcemarketing.com/).

El proyecto nació como una remodelación y actualización del sitio web del taller. Tiene una duración estimada de 8 meses con 3 etapas: diseño, desarrollo e implementación.
